# 玄机 IDE v1.0.0-alpha.0 Release Notes

## 概述

这是玄机 IDE 的首个 Alpha 内部测试版本，基于 VS Code 1.112.0 内核构建。

## 主要特性

### 品牌化定制
- 全新品牌标识："玄机" (XuanJi)
- 自定义窗口标题、关于页面、安装器品牌
- 独立的应用标识符和数据目录 (`.xuanji`)

### 多平台支持
- **Windows**: Inno Setup 安装器 (.exe)
- **macOS**: DMG 镜像 (.dmg)
- **Linux**: deb 包 + AppImage

### 扩展市场
- 集成 Open VSX 开放扩展市场
- 支持搜索、安装、管理扩展

### CI/CD
- GitHub Actions 自动化构建流水线
- 三平台并行编译与打包
- 自动化发布工作流

## 已知限制

- 安装包未经代码签名（Windows SmartScreen 警告、macOS 需手动允许）
- 不支持自动更新
- Rust sidecar 桥接功能开发中
- 部分 VS Code 专有扩展在 Open VSX 上不可用
